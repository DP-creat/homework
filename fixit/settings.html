HUD

OPERATIONAL MANUAL

// ARC-9 SYSTEM INTERFACE L2

SECTION 01 // CORE NAVIGATION

CORE SCANNER: Центральный био-монитор.
• 1-й клик: Активация интерфейса (READY).
• 2-й клик: Запуск CHRONO (Секундомер). Кольцо начинает вращение.
• 3-й клик: Сброс и возврат к REMAINING RESOURCE (ресурс дня до 23:00).

LASER CROSS (Вверху): Глубокая очистка текущего цикла (DEEP RESET). Обнуляет счетчики всех активных юнитов без удаления самих ячеек.

SECTION 02 // UNIT CONTROL (BATTERY CELL)

Каждый юнит имеет два протокола защиты:

КЛИК 1 (STASIS): Пробуждение юнита (AWAKE). Ячейка набирает яркость, активируется "часовой пояс" с точками предыдущих приемов.

КЛИК 2 (DEPLOY): Фиксация приема. Увеличивает счетчик, заполняет круговой индикатор прогресса и ставит метку в логи.

UPPER ICONS (X): Доступны только в режиме AWAKE.
• Зеленый X (Справа): Откат последнего приема (-1).
• Красный X (Слева): Полное удаление юнита из системы.

SECTION 03 // DATA TERMINALS

LAVA BUTTONS: Управление сервисными панелями. Разворачивают формы создания кликеров (CLICKER CRAFT) и записи тренировок (KINETIC TARGETS).

HUD LOG (Нижняя панель): Терминал истории. Нажмите на лазерную линию для развертывания вверх. Позволяет выделять и копировать данные без сворачивания окна.

NUCLEAR RESET (Footer): Экстренное стирание. Удаляет ВСЕ данные, включая конфигурации созданных юнитов и LocalStorage.
